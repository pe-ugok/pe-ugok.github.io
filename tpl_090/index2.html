1.　Panel design





/　Clothes Choice　/


Arduinoを用いて“忙しい朝でもコップを置いたら朝食を食べている間にコーディネートが完成する”というテーマで制作。
パネルイラストと小物作りを担当しました。



□　パネルイラスト



□　小物



| 　デザイン 　|


作品の使い方をスピーディに伝え、遠くからでも目に留まるように、インパクトのある四コマ漫画で表現しました。
朝が忙しく、毎日のコーディネートを考えるのが面倒くさいと考える10～20代の女性をターゲットとしているため、全体的にポップな配色にして可愛らしさを表現しました。
朝食のシーンを再現するために用いた小物のパンは、紙粘土を成形・着色し、レジンで蜂蜜を表現しました。



|　使用ツール　|

2.　Logo & Header





/　UGOK logo & header　/


“UGOK（融合系女子）”としてゼミ活動をSNSで発信する際に使用するロゴとヘッダーを制作しました。



□　Twitter画面



| 　デザイン 　|


親近感・身近さを感じてもらえるように、丸く太めの線の文字を作り、パステル調のカラーを使うことで可愛らしくなるようにしました。UGOKは「うごく」と読むこともできるので「GO」という文字、また活動の総称である「letusプロジェクト」には活動内容のアピール+その内容に興味を持った人からの依頼もお待ちしております！という意味があるので「OK」という文字、そしてその二つの文字に共通する“O”を重ねることで融合しているというイメージにしました。



|　使用ツール　|


illustrator　/　CLIP STUDIO PAINT

3.　Web application





/　GYAKUSAN SCHEDULE　/


目的地への到着時間や自分が身支度する時間を入力すると、出発までのスケジュールを逆算してくれるアプリを作りました。
GoogleMapのAPIを取り入れ、入力した地点間の移動時間を取得できるようにしました。



□　車verの入力画面



| 　デザイン 　|


誰でも使いやすいようにシンプルなデザインを意識しました。
また、視覚的にわかりやすいように、電車・車と交通手段別に配色を統一。
数値や文字の入力の際には、ステップごとに説明を加えることで簡単に入力できるようにしました。



|　使用ツール　|


vue.js　/　html　/　css　/　illustrator
